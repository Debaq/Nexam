# Core

Funcionalidades core del sistema.

## Módulos

### storage/
- Configuración de IndexedDB con Dexie
- Schemas de datos
- Migraciones

### vision/
- Pipeline de Computer Vision
- Integración OpenCV.js
- Integración ONNX Runtime
- Utilidades de procesamiento de imágenes

### export/
- Exportadores (PDF, ZIP, Excel)
- Parsers (GIFT)
- Utilidades de archivo

## Uso

Estos módulos son utilizados por todos los features del sistema.
