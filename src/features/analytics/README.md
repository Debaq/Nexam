# Analytics Feature

Análisis y reportes de evaluaciones.

## Estructura

- `components/` - Gráficos, dashboards, reportes
- `hooks/` - Cálculos estadísticos
- `services/` - Generación de reportes
- `types/` - Analytics, Report types
- `styles/` - Visualizaciones

## Funcionalidades

- Estadísticas por evaluación
- Análisis por pregunta
- Rendimiento de estudiantes
- Gráficos y visualizaciones
- Export de reportes (PDF, Excel)
- Identificación de preguntas difíciles

## Uso

Implementación en fases posteriores al MVP.
