# Questions Feature

Gestión del banco de preguntas del sistema.

## Estructura

- `components/` - Componentes React para UI de preguntas
- `hooks/` - Custom hooks para lógica de preguntas
- `services/` - Servicios de negocio (CRUD, validaciones)
- `types/` - TypeScript definitions para Question
- `styles/` - CSS específico del módulo

## Funcionalidades

- CRUD completo de preguntas
- Soporte múltiples tipos (selección múltiple, verdadero/falso, desarrollo)
- Categorización y etiquetado
- Niveles de dificultad
- Estadísticas de uso
- Import/Export GIFT y ZIP
- Soporte de imágenes en preguntas
