# Correction Feature

Corrección automática con Computer Vision.

## Estructura

- `components/` - UI de corrección, visor de resultados
- `hooks/` - Pipeline de procesamiento
- `services/` - OpenCV, YOLO, validación
- `types/` - CorrectionResult, DetectedMark types
- `styles/` - Visualización de correcciones

## Funcionalidades

- Pipeline completo de Computer Vision
- Preprocesamiento con OpenCV.js
- Detección de marcas con YOLOv11n
- Alineación de imágenes
- Extracción de respuestas
- Validación automática
- Cálculo de puntajes
