# Exams Feature

Generación y gestión de evaluaciones.

## Estructura

- `components/` - Builder de exámenes, configuración
- `hooks/` - Lógica de generación y aleatorización
- `services/` - Generación de PDFs, códigos únicos
- `types/` - Exam, ExamConfig types
- `styles/` - Estilos del generador

## Funcionalidades

- Builder interactivo de exámenes
- Configuración de aleatorización
- Generación de PDFs (pruebas + hojas respuesta)
- Sistema de códigos únicos (4 letras)
- Tipos de evaluación (uniforme/diferenciada)
- Asignación de estudiantes

## Uso

Ver `docs/EXAMS.md` para implementación detallada.
