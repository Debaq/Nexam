# Students Feature

Gestión de estudiantes.

## Estructura

- `components/` - Formularios, lista de estudiantes
- `hooks/` - Validación RUT, búsqueda
- `services/` - CRUD estudiantes
- `types/` - Student type
- `styles/` - UI de gestión

## Funcionalidades

- CRUD completo de estudiantes
- Validación de RUT chileno
- Búsqueda y filtros
- Import/Export de nóminas
- Asignación a evaluaciones

Ver `docs/EXAMS.md` para implementación detallada.
